╔══════════════════════════════════════════════════════════════╗
║  Optional: Virtualization ON (advanced)                      ║
╚══════════════════════════════════════════════════════════════╝

  1. PC restart → BIOS (F2 / F10 / Del key - laptop pe likha hota hai)
  2. Find: Intel VT-x / AMD-V / Virtualization → Enabled
  3. Windows: Turn Windows features on →
     - Virtual Machine Platform
     - Windows Subsystem for Linux (WSL)
  4. Restart

  Company laptop ho to IT admin se pucho — shayad policy se band ho.

  Production server par baad mein PostgreSQL + Docker use kar sakte ho.
  Abhi local testing ke liye SQLite kaafi hai.
